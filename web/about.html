STOCK MANAGEMENT SYSTEM

©CREATED BY MOSOUWER JAMIL


Stock Management System is Dynamic Web Application capable of managing stocks in inventory . stocks can be added and the price can be updated in existing stocks . I have also added the Sales functionality of the stocks in the inventory firstly stocks are needed to be added in the cart then further billing can be done.

This web Application is developed in Java platform. this web application uses the technolgies like Servlet , Java Server Pages as the backend and HTML5 and CSS3 as the frontend and it also uses MySQl as the Database

The login mechanism is secured with Http Sessions , Welcome JSP keeps on cheking the Session status at the interval of 10 seconds and if there is an unusual activity then it invalidate the Session and redirected to the login page.
